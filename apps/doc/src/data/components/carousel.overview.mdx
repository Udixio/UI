## Usage

Use Carousel with CarouselItem children to create a horizontally scrollable showcase.

```bash
import { Carousel, CarouselItem } from "@udixio/ui-react"
```

```tsx
<Carousel variant="hero" gap={8}>
  <CarouselItem>
    <div className="p-6 bg-surface rounded-xl">Slide 1</div>
  </CarouselItem>
  <CarouselItem>
    <div className="p-6 bg-surface rounded-xl">Slide 2</div>
  </CarouselItem>
  <CarouselItem>
    <div className="p-6 bg-surface rounded-xl">Slide 3</div>
  </CarouselItem>
</Carousel>
```

## Variants

Available variants: `hero`, `center-aligned hero`, `multi-browse`, `un-contained`, `full-screen`.

```tsx
<div className="space-y-6">
  <Carousel variant="hero" gap={12}>
    <CarouselItem><div className="p-6 bg-surface rounded-xl">Hero</div></CarouselItem>
    <CarouselItem><div className="p-6 bg-surface rounded-xl">Hero</div></CarouselItem>
  </Carousel>
  <Carousel variant="multi-browse" gap={8}>
    <CarouselItem><div className="p-4 bg-surface rounded-xl">Multi</div></CarouselItem>
    <CarouselItem><div className="p-4 bg-surface rounded-xl">Browse</div></CarouselItem>
    <CarouselItem><div className="p-4 bg-surface rounded-xl">Items</div></CarouselItem>
  </Carousel>
</div>
```

## Controlling item width

You can set a fixed item width using the `width` prop on CarouselItem.

```tsx
<Carousel variant="multi-browse" gap={8}>
  <CarouselItem width={260}><div className="p-4 bg-surface rounded-xl">260px</div></CarouselItem>
  <CarouselItem width={260}><div className="p-4 bg-surface rounded-xl">260px</div></CarouselItem>
  <CarouselItem width={260}><div className="p-4 bg-surface rounded-xl">260px</div></CarouselItem>
</Carousel>
```

## Behavior and callbacks

- `gap`: spacing between items.
- `marginPourcent`: outer margins as a percentage of the viewport width.
- `scrollSensitivity`: adjust scroll/drag responsiveness.
- `onChange`: called with the active index.

```tsx
<Carousel
  variant="hero"
  gap={16}
  marginPourcent={12}
  scrollSensitivity={0.8}
  onChange={(index) => console.log('Active slide:', index)}
>
  {[1,2,3,4].map(i => (
    <CarouselItem key={i}>
      <div className="p-6 bg-surface rounded-xl">Slide {i}</div>
    </CarouselItem>
  ))}
</Carousel>
```

## Sizing behavior

Items adapt their width based on the variant settings. Tune presentation via `gap` and `variant`.

```tsx
<Carousel variant="hero" gap={16} />
```
