## Usage

Import the Card component from @udixio/ui-react to group related content and actions.

```bash
import { Card } from "@udixio/ui-react"
```

```tsx
<Card>
  <div className="p-4 space-y-2">
    <h3 className="text-lg font-semibold">Card title</h3>
    <p>Card body content goes here.</p>
  </div>
</Card>
```

## Variants

```jsx
<div className="flex flex-col gap-4">
  <Card>
    <div className="p-4">Outlined (default)</div>
  </Card>
  <Card variant="elevated">
    <div className="p-4">Elevated</div>
  </Card>
  <Card variant="filled">
    <div className="p-4">Filled</div>
  </Card>
</div>
```

## Interactive state

Enable hover and press feedback by setting `isInteractive` to true. Use it for clickable cards (e.g., navigation tiles).

```jsx
<Card isInteractive>
  <div className="p-4">Clickable card</div>
</Card>
```

## Composition patterns

Cards are layout containers. Combine simple elements to create common patterns.

- Header + body

```jsx
<Card>
  <div className="p-4 space-y-1">
    <h3 className="text-base font-semibold">Project Aurora</h3>
    <p className="text-sm text-on-surface-variant">Last updated 2 days ago</p>
  </div>
</Card>
```

- Media + text

```jsx
<Card>
  <img className="w-full h-40 object-cover" src="https://picsum.photos/640/240" alt="Cover" />
  <div className="p-4">
    <p>Beautiful landscapes collection</p>
  </div>
</Card>
```

- Actions

```jsx
<Card>
  <div className="p-4 space-y-3">
    <p>Are you sure you want to delete this item?</p>
    <div className="flex gap-2">
      <Button label="Cancel" variant="text" />
      <Button label="Delete" variant="filled" />
    </div>
  </div>
</Card>
```

## Grid of cards

Display multiple cards using your layout system.

```jsx
<div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-4">
  {[1,2,3].map(i => (
    <Card key={i} variant={i % 2 ? 'outlined' : 'elevated'}>
      <div className="p-4">Card {i}</div>
    </Card>
  ))}
</div>
```
